## Narrative

On application startup, the **Load Saved Character Page** is displayed. From this page, the user may either load an existing character or create a new character.

Regardless of which option is selected, the **Character Creation Page** is then displayed. If a new character is being created, all fields on the page are initially blank. If an existing character is loaded, the fields are populated with that character’s stored information.

When creating a new character, the user may enter a character name, input values for ability scores, and select the character’s species and heroic class. The user may also select feats, skills, and talents for the character.

Several character statistics are automatically calculated by the system. These include ability modifiers, defenses, hit points, damage threshold, speed, Force points, Destiny points, skill bonuses, base attack bonus, and attack bonus.

Languages are automatically listed based on the character’s species. If the character is eligible for additional languages, the user may select bonus languages.

The user may enter the character’s equipment. If eligible, the user may also select Force powers.

All characters created in the prototype are **Level 1 characters**.

The Character Creation Page also includes controls for managing characters. Buttons located in the top-right area of the character sheet allow the user to save the current character, delete the current character, or load another saved character.
